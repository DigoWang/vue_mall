# vue_phone_store
基于Vue全家桶（vue2.0 + vue-router + vuex）+ vantui + less的商城系统前台

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
